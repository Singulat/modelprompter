# Developing
## Requirements
- git - https://git-scm.com/downloads
- NodeJS - https://nodejs.org/en/download/

## Setup
```bash
# Clone this project locally
git clone https://github.com/0xozram/modelprompter
cd modelprompter

# Install dependencies
npm install

# Start a local server with live reload
npm start

# Stop the server
# CTRL+C
```
